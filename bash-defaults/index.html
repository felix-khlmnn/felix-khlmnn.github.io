bash-defaults

Link to repository

My default settings for the Bourne-Again-Shell, focusing on integrating key bindings as of right now.

The .bashrc file is only slightly modified to use the source command on a file similar to the optional .bash_aliases file used for defining aliases.

Usage

If you decide to clone this repository, you can find the part of .bashrc that loads .bash_bindings in to inject it into your already existing .bashrc file.

Optionally, you can completely replace the .bashrc if you don't have any changes that you'd like to preserve.

Default bindings

The bindings I have included as of right now are:

  • Alt-D: Disowns the command. Allows you to use the shell while a command is being executed.

  • Alt-C: Clears the shell immediately, without having to press enter.


Modify it as you like :)
